## Overview

StudentStarterPlus helps students discover opportunities (internships, events, programs) while enabling organizations to reach their target audience. Built with role-based authentication for students, organizations, and admins.

## Tech Stack

- **Frontend:** React, Tailwind CSS (deployed on Vercel)
- **Backend:** Node.js, Express.js (deployed on Render)
- **Database:** PostgreSQL via Supabase
- **Auth:** Auth0
- **APIs:** 12+ RESTful endpoints

## Features

- Role-based authentication (students, organizations, admins)
- Organization verification and approval workflow
- Opportunity posting and application system
- User profile management
- Admin dashboard
